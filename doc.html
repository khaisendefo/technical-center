Письменные уведомления и консультации

Для получения любой письменной консультации и регистрации обращения по любой программе гарантии, клиенту необходимо подтвердить свое участие в программе гарантии. Для этого, пользователю необходимо предоставить (прислать) копии следующих документов на адрес электронной почты mail@cgarant.com:
- копия сертификата программы гарантии
- копия опционного или агентского договора
- копия акта о подключении к программе гарантии
- копия свидетельства о регистрации ТС (с 2-х сторон)
- фотография текущего пробега ТС, подключенного к программе гарантии
После подтверждения подключения пользователя к программе гарантии, получения информации и копий всех перечисленных документов, возможно составление любого письменного уведомления по всем вопросам, связанным с программами гарантий.

Как выбрать актуальную версию Правил?

Все размещенные на сайте документы (Правила и условия программ) имеют номер версии и дату обновления.

Для того, чтобы выбрать для ознакомления или печати актуальный документ, необходимо сравнить номер сертификата с маской номеров (идентификатор). Например, если Сертификат имеет номер FE-05289 (идентификатор FE), значит, этот сертификат выдан участнику программы гарантии "ФЕНИКС" (маска номера Сертификата по программе "ФЕНИКС": FE-*****).

Далее следует проверить дату выдачи Сертификата и выбрать версию правил, актуальную в этот момент времени. Например, если по префиксу стало понятно, что Сертификат выдан автовладельцу, подключенному к программе Гарантии "ФЕНИКС" (например, FE-05387) и выдан Сертификат 05.06.2022 г., то для данного Сертификата действуют правила Гарантии "ФЕНИКС" в версии V5.0 от 01.06.2022. Если Сертификат выдан ранее даты 01.06.2022 г. (к примеру, 10.05.2022), то для данного Сертификата действуют правила Гарантии "ФЕНИКС" в версии V4.0.

Если у вас имеются трудности с выбором актуальной версии Правил для имеющегося Сертификата, заполните форму для заказа рассылки изменений в Правилах оказания услуг по программам и актуальная для Вас версия будет выслана на указанный в форме электронный адрес.

В Правила оказания услуг по Гарантийным программам могут быть внесены изменения.

При возникновении у клиента каких-либо трудностей, связанных с выбором для ознакомления актуальной версии правил какой-либо из программы гарантии, клиенту необходимо связаться по указанному на сайте телефону для получения прямой ссылки на Правила по его сертификату и/или договору.

Для получения консультации по любой программе гарантии к которой подключен Клиент, ему необходимо предоставить подтверждение подключения в виде копий сертификата и свидетельства о регистрации ТС на электронный адрес Компании.
